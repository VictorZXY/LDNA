\section{Setup}
\label{sec:expr-setup}

We fix notation and state the two background facts the proofs rely on. An
attributed graph is $\mathcal{G}=(\mathcal{V},\mathcal{E})$ with node-feature
matrix $\mathbf{X}$, adjacency matrix $\mathbf{A}$, and optional edge-feature
matrix $\mathbf{E}$; row $\mathbf{x}_u$ is the feature of node $u\in\mathcal{V}$,
and $\mathbf{e}_{v,u}$ is the feature of edge $(v,u)\in\mathcal{E}$. We write
$\mathcal{N}_u=\{v\mid(v,u)\in\mathcal{E}\}$ for the neighbourhood of $u$ and
$n=|\mathcal{V}|$. A message-passing GNN (MP-GNN) \citep{Gilmer2017MPNN} carries
node embeddings $\mathbf{h}_u^{(l)}$, updates them by combining each node with a
\emph{permutation-invariant} aggregate of its neighbour messages, and finally
reads out the multiset $\{\!\{\mathbf{h}_u^{(L)}\mid u\in\mathcal{V}\}\!\}$ into a
graph embedding $\mathbf{h}_{\mathcal{G}}$.

Throughout, node and edge feature alphabets are finite and graph sizes are
bounded by $n\le N$. All expressiveness statements assert the existence of
parameters realising a behaviour, in the sense of \cref{def:expr-power}.

\paragraph{The reference test.}
The 1-WL colour refinement (with edge colours) assigns
\begin{equation}
  c_u^{(0)}=\mathbf{x}_u,\qquad
  c_u^{(l+1)}=\operatorname{HASH}\!\Big(c_u^{(l)},\ \{\!\{\,c_v^{(l)}:v\in\mathcal{N}_u\,\}\!\}\Big),
  \label{eqn:expr-wl}
\end{equation}
with $\operatorname{HASH}$ injective and colours ranging over a finite set at
each round; for the edge-coloured variant the neighbour term is
$\{\!\{(c_v^{(l)},\mathbf{e}_{v,u}):v\in\mathcal{N}_u\}\!\}$. Two graphs are
\emph{1-WL-indistinguishable} if their colour histograms
$\{\!\{c_u^{(l)}\mid u\in\mathcal{V}\}\!\}$ agree at every round $l$; this forces
equal node counts.

\begin{definition}[Distinguishing power]
\label{def:expr-power}
A model class $\mathcal{M}$ (ranging over its parameters) \emph{distinguishes} a
pair $(\mathcal{G}_1,\mathcal{G}_2)$ if some parameters give
$\mathcal{M}(\mathcal{G}_1)\ne\mathcal{M}(\mathcal{G}_2)$. It is \emph{at least as
expressive as} 1-WL if it distinguishes every pair that 1-WL distinguishes, and
\emph{strictly more expressive} than 1-WL if it additionally distinguishes some
pair that 1-WL does not.
\end{definition}

\begin{fact}[Finite-domain realisability]
\label{fact:expr-finite}
Let the relevant feature and colour sets be finite.
\begin{enumerate}
  \item[(a)] A learnable family such as an MLP can realise any prescribed
        function on a finite domain.
  \item[(b)] There is an encoding of a finite set whose summation over any
        multiset of bounded size is injective in that multiset.
\end{enumerate}
We use these only as stated; \citet[Lemma~5]{Xu2019GIN} and
\citet{Zaheer2017DeepSets} establish (b), and (a) is standard.
\end{fact}

\subsection{The LDNA aggregation, made precise}
\label{sec:expr-ldna}

LDNA replaces the permutation-invariant aggregate of an MP-GNN by a learnable
map applied to the neighbour messages after they are placed into fixed slots by a
canonical order. We write it with three learnable functions --- a message map
$\phi_{\bm{\Theta}}$, an aggregation map $\psi_{\bm{\theta}}$ and an update map
$\gamma_{\bm{\Theta}}$ (all realisable as MLPs, with parameters $\bm{\Theta}$,
$\bm{\theta}$) --- and a canonical sorting permutation
$\pi^*:\mathcal{V}\to\{1,\dots,n\}$.

Each directed edge $(v,u)$ carries a message
\begin{equation}
  \mathbf{m}_{v,u}^{(l+1)}=
  \begin{cases}
    \phi_{\bm{\Theta}}^{(l+1)}\!\big(\mathbf{h}_u^{(l)},\mathbf{h}_v^{(l)}\big) & \text{(LDNA),}\\[2pt]
    \phi_{\bm{\Theta}}^{(l+1)}\!\big(\mathbf{h}_u^{(l)},\mathbf{h}_v^{(l)},\mathbf{e}_{v,u}\big) & \text{(LDNA-e).}
  \end{cases}
  \label{eqn:expr-message}
\end{equation}
The messages of $u$ are gathered into a fixed-size matrix
$\mathbf{S}_u^{(l+1)}\in\mathbb{R}^{N\times d_h}$ by scattering each neighbour's
message to the row indexed by its canonical position: for $p=1,\dots,N$,
\begin{equation}
  \big(\mathbf{S}_u^{(l+1)}\big)_p=\sum_{v\in\mathcal{N}_u}\mathbb{1}\!\left[\pi^*(v)=p\right]\,\mathbf{m}_{v,u}^{(l+1)} .
  \label{eqn:expr-stack}
\end{equation}
Because $\pi^*$ is injective, each row receives at most one message; rows whose
position is not that of a neighbour are $\mathbf{0}$. The row index is the
neighbour's \emph{global} canonical position, so reading a row recovers which
node contributed it --- the identity information that a permutation-invariant
aggregate discards. The node is then updated from its own state and the
aggregated slots, and after $L$ layers a readout pools the node embeddings:
\begin{align}
  \mathbf{h}_u^{(l+1)} &= \gamma_{\bm{\Theta}}^{(l+1)}\!\Big(\mathbf{h}_u^{(l)},\ \psi_{\bm{\theta}}^{(l+1)}\!\big(\mathbf{S}_u^{(l+1)}\big)\Big),
  \label{eqn:expr-update}\\
  \mathbf{h}_{\mathcal{G}} &= \operatorname{READOUT}\big(\{\!\{\mathbf{h}_u^{(L)}\mid u\in\mathcal{V}\}\!\}\big).
  \label{eqn:expr-readout}
\end{align}
The update \labelcref{eqn:expr-update} keeps $\mathbf{h}_u^{(l)}$ as an explicit
self-argument.

\begin{definition}[Canonical labelling]
\label{def:expr-canonical}
A \emph{canonical labelling} assigns to each graph $\mathcal{G}$ a bijection
$\pi^*_{\mathcal{G}}:\mathcal{V}\to\{1,\dots,n\}$ that is isomorphism-invariant at
the labelled-graph level: for every isomorphism $f:\mathcal{G}_1\to\mathcal{G}_2$,
relabelling by $\pi^*_{\mathcal{G}_1}$ and $\pi^*_{\mathcal{G}_2}$ yields the same
ordered attributed graph. Such a labelling is obtainable by completing a
feature-based sort through individualization--refinement \citep{McKay2014Nauty},
with quasi-polynomial worst-case cost \citep{Babai2016GI}.
\end{definition}

\section{LDNA is strictly more expressive than 1-WL}
\label{sec:expr-main}

\begin{theorem}
\label{thm:expr-strict}
If $\pi^*$ is induced by a canonical labelling (\cref{def:expr-canonical}), then
LDNA (\cref{eqn:expr-message,eqn:expr-stack,eqn:expr-update,eqn:expr-readout}) is
strictly more expressive than the 1-WL test, and hence than any MP-GNN with
permutation-invariant aggregation \citep{Xu2019GIN,Morris2019WL}. The same holds
for LDNA-e against the edge-coloured 1-WL refinement.
\end{theorem}

We prove the theorem through two lemmas: LDNA can simulate 1-WL
(\cref{lem:expr-simulate}), and LDNA separates a 1-WL-indistinguishable pair
(\cref{lem:expr-separate}).

\begin{lemma}[Simulation]
\label{lem:expr-simulate}
LDNA is at least as expressive as 1-WL.
\end{lemma}

\begin{proof}
\emph{Claim.} There exist parameters and, for each layer $l$, an injective map
$\alpha_l$ such that $\mathbf{h}_u^{(l)}=\alpha_l\big(c_u^{(l)}\big)$ for every
node $u$ of every graph of size $\le N$.

\paragraph{Base ($l=0$).}
$c_u^{(0)}=\mathbf{x}_u$, and taking the input encoding to be injective on the
finite feature alphabet (\cref{fact:expr-finite}(a)) gives
$\mathbf{h}_u^{(0)}=\alpha_0(\mathbf{x}_u)$ with $\alpha_0$ injective.

\paragraph{Inductive step.}
Assume the claim at layer $l$; we realise it at layer $l+1$.
\begin{enumerate}
  \item[(1)] \emph{Message.} Since $\alpha_l$ is injective and all arguments
        range over finite sets, $\phi_{\bm{\Theta}}^{(l+1)}$ can encode the pair
        $(\mathbf{h}_v^{(l)},\mathbf{e}_{v,u})$ --- equivalently
        $(c_v^{(l)},\mathbf{e}_{v,u})$ --- injectively into
        $\mathbf{m}_{v,u}^{(l+1)}$ (\cref{fact:expr-finite}(a)).
  \item[(2)] \emph{Aggregation.} Let $\psi_{\bm{\theta}}^{(l+1)}$ ignore the slot
        positions and sum the rows of $\mathbf{S}_u^{(l+1)}$; by
        \labelcref{eqn:expr-stack} this equals
        $\sum_{v\in\mathcal{N}_u}\mathbf{m}_{v,u}^{(l+1)}$. With the message
        encoding of step~(1) chosen summation-injective
        (\cref{fact:expr-finite}(b)), the value
        $\psi_{\bm{\theta}}^{(l+1)}(\mathbf{S}_u^{(l+1)})$ is then an injective
        function of the neighbour multiset
        \begin{equation*}
          \{\!\{(c_v^{(l)},\mathbf{e}_{v,u})\mid v\in\mathcal{N}_u\}\!\}.
        \end{equation*}
  \item[(3)] \emph{Update.} The self-argument $\mathbf{h}_u^{(l)}$ of
        $\gamma_{\bm{\Theta}}^{(l+1)}$ lets it injectively combine $c_u^{(l)}$
        with the aggregate (\cref{fact:expr-finite}(a)); an empty neighbourhood
        is harmless: the empty sum is, by \cref{fact:expr-finite}(b), distinct
        from every non-empty aggregate, and the self-argument determines the
        output.
\end{enumerate}
Composing (1)--(3), $\mathbf{h}_u^{(l+1)}$ is an injective function of
$\big(c_u^{(l)},\{\!\{c_v^{(l)}\mid v\in\mathcal{N}_u\}\!\}\big)$, which by
injectivity of $\operatorname{HASH}$ in \labelcref{eqn:expr-wl} determines
$c_u^{(l+1)}$; set $\alpha_{l+1}$ accordingly, proving the claim.

\paragraph{Graph level.}
Take the readout to sum a summation-injective encoding of the final embeddings
(\cref{fact:expr-finite}(b)); by the claim it is injective in the colour
histogram $\{\!\{c_u^{(L)}\}\!\}$. Hence whenever 1-WL assigns two graphs
different colour histograms at some round, LDNA (run to that depth) assigns them
different graph embeddings. The edge-coloured variant is identical with
$\mathbf{e}_{v,u}$ carried through step~(1).
\end{proof}

\begin{lemma}[Separation]
\label{lem:expr-separate}
There is a 1-WL-indistinguishable pair of graphs that LDNA distinguishes.
\end{lemma}

\begin{proof}
Fix $m\ge 3$ and let $\mathcal{G}_1=C_{2m}$ be the cycle on $2m$ nodes and
$\mathcal{G}_2=C_m\sqcup C_m$ the disjoint union of two $m$-cycles, both with a
single constant node feature and no edge features.

\paragraph{Step 1: the pair is 1-WL-indistinguishable.}
Both graphs are $2$-regular on $2m$ nodes with identical initial colours. By
induction on $l$, all nodes share one colour at every round: if every node has
colour $a$ at round $l$, each has neighbour multiset $\{\!\{a,a\}\!\}$, so
\labelcref{eqn:expr-wl} returns a single colour at round $l+1$. The colour
histograms therefore coincide at every round, so $\mathcal{G}_1$ and
$\mathcal{G}_2$ are 1-WL-indistinguishable.

\paragraph{Step 2: the canonical labelling.}
A canonizer may fix any one representative labelling per isomorphism class. For
$C_{2m}$ take the cyclic labelling on $\{1,\dots,2m\}$, so that position $p$ is
adjacent to positions $p\pm1$ modulo $2m$. For $C_m\sqcup C_m$ take the two
consecutive blocks
\begin{equation*}
  B_1=\{1,\dots,m\},\qquad B_2=\{m+1,\dots,2m\},
\end{equation*}
each of the two $m$-cycles cyclically labelled within its own block.

\paragraph{Step 3: separating parameters, algebraically.}
Take $\phi_{\bm{\Theta}}\equiv\mathbf{1}$ on a single channel, so every message
equals $1$ and, by \labelcref{eqn:expr-stack}, $\mathbf{S}_u$ is the indicator
vector of the \emph{slot set}
\begin{equation*}
  P_u:=\{\pi^*(v)\mid v\in\mathcal{N}_u\}\subseteq\{1,\dots,2m\} .
\end{equation*}
Now compare the two labellings. In $\mathcal{G}_2$ every edge joins two nodes of
the same block, so every $P_u$ lies inside a single block. In $\mathcal{G}_1$ the
node at position $1$ has neighbours at positions $2$ and $2m$, giving
\begin{equation*}
  P=\{2,\,2m\},\qquad 2\in B_1,\quad 2m\in B_2 ,
\end{equation*}
using $2\le m$ and $2m\ge m+1$ (both hold for $m\ge 3$); this $P$ meets both
blocks. Consequently the multiset $\{\!\{P_u\mid u\in\mathcal{V}\}\!\}$ of
$\mathcal{G}_1$ contains a set contained in no single block, whereas that of
$\mathcal{G}_2$ contains none, so the two multisets differ. Let
$\psi_{\bm{\theta}}$ realise an injective map on the finite domain of indicator
vectors (\cref{fact:expr-finite}(a)); then $\psi_{\bm{\theta}}(\mathbf{S}_u)$ is
injective in $P_u$. Let $\gamma_{\bm{\Theta}}$ be injective; since
$\mathbf{h}_u^{(0)}$ is the same constant in both graphs, one layer yields
$\mathbf{h}_u^{(1)}$ as an injective function of $P_u$, so
$\{\!\{\mathbf{h}_u^{(1)}\}\!\}$ is the image of $\{\!\{P_u\}\!\}$ under an
injective map --- and these image multisets differ across $\mathcal{G}_1$ and
$\mathcal{G}_2$.

\paragraph{Step 4: conclusion.}
Taking the readout injective (\cref{fact:expr-finite}(b)), the two graphs receive
different embeddings; since 1-WL cannot separate them, LDNA distinguishes a
1-WL-indistinguishable pair. The smallest instance is $m=3$: $C_6$ versus two
triangles.
\end{proof}

\begin{proof}[Proof of \cref{thm:expr-strict}]
\cref{lem:expr-simulate} gives $\text{LDNA}\succeq\text{1-WL}$ and
\cref{lem:expr-separate} gives a 1-WL-indistinguishable pair that LDNA separates;
by \cref{def:expr-power} LDNA is strictly more expressive than 1-WL, and hence
than any MP-GNN with permutation-invariant aggregation, whose power is bounded by
1-WL \citep{Xu2019GIN,Morris2019WL}. The LDNA-e statement follows from the same
two lemmas against the edge-coloured refinement.
\end{proof}

\begin{proposition}[The canonical labelling is necessary]
\label{prop:expr-necessity}
The separation in \cref{thm:expr-strict} depends on $\pi^*$ being a canonical
labelling.
\begin{enumerate}
  \item[(i)] If the stacked input \labelcref{eqn:expr-stack} is replaced by the
        content-sorted list of the messages (ties left unbroken), then LDNA is
        bounded by 1-WL, and by \cref{lem:expr-simulate} is exactly 1-WL.
  \item[(ii)] If $\pi^*$ is obtained by sorting on isomorphism-invariant per-node
        quantities, it individualizes no node of a vertex-transitive graph, so on
        the classical 1-WL-hard pairs it supplies no identities.
\end{enumerate}
\end{proposition}

\begin{proof}
\emph{(i).} Sorting the messages by content is a bijection from the multiset
$\{\!\{\mathbf{m}_{v,u}\mid v\in\mathcal{N}_u\}\!\}$ to a sorted tuple, so
$\psi_{\bm{\theta}}\circ\mathrm{sort}$ is a function of that multiset alone,
i.e.\ a permutation-invariant aggregate. LDNA is then an MP-GNN with
permutation-invariant aggregation, whose power is bounded by 1-WL
\citep{Xu2019GIN,Morris2019WL}; with \cref{lem:expr-simulate} --- whose
aggregation is the order-agnostic summation, hence available under
content-sorting unchanged --- it is exactly 1-WL.

\emph{(ii).} Let $\iota$ be an isomorphism-invariant node quantity (raw features,
degree, 1-WL colour, subgraph counts, and the like), and let $\pi^*$ sort by
$\iota$. If $\mathcal{G}$ is vertex-transitive, then for any two nodes $u,w$ some
automorphism maps $u$ to $w$, whence $\iota(u)=\iota(w)$; thus $\iota$ assigns
every node the same sort key, so it induces no individualizing invariant order,
and any bijection $\pi^*$ extending it is determined entirely by tie-breaking. On a vertex-transitive 1-WL-indistinguishable pair --- the $4\times4$
rook's graph versus the Shrikhande graph, or the CSL family --- such an order
therefore supplies no identities, and the construction of
\cref{lem:expr-separate} is unavailable. Breaking the ties arbitrarily makes the
output depend on the input labelling, destroying well-definedness. A canonical
labelling (\cref{def:expr-canonical}) individualizes the nodes while remaining
invariant at the labelled-graph level, and is the only way to secure both.
\end{proof}

\begin{remark}[Random initialization]
\label{rem:expr-random-init}
With the parameters of $\phi_{\bm{\Theta}}$, $\psi_{\bm{\theta}}$,
$\gamma_{\bm{\Theta}}$ drawn independently from a continuous distribution, each
layer's maps are injective almost surely: over the finite domain of colours, the
parameter settings that collapse two distinct colours form a finite union of
measure-zero sets. Composing, the embeddings $\mathbf{h}_u^{(l)}$ refine the 1-WL
colours $c_u^{(l)}$ almost surely, and the same genericity makes the readout
injective on the refined histograms, so a randomly initialized LDNA is at least
as expressive as 1-WL without any training. The strict separation of
\cref{lem:expr-separate} instead uses the explicit deterministic parameters
constructed there. This differs from random node initialization
\citep{Abboud2021RNI}, which randomizes the inputs rather than the aggregator.
\end{remark}

\begin{remark}[Permutation invariance]
\label{rem:expr-invariance}
Although $\pi^*$ imposes an order, the LDNA output is a deterministic graph
invariant: by \cref{def:expr-canonical} the labelling depends only on the
isomorphism class, and any choice of representative within an automorphism orbit
yields the same labelled graph --- hence the same stacked inputs $\mathbf{S}_u$
and the same embeddings. The sort that secures permutation invariance and the
labelling that supplies node identities are one and the same object.
\end{remark}
