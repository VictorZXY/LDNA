\section{Proof of Theorem 1}
\label{sec:expr-proof}

Throughout, node and edge feature alphabets are finite and graph sizes are
bounded by $N$; every statement asserts the existence of parameters realising a
behaviour, in the sense of \cref{def:expressiveness}. We write $\{\!\{\cdot\}\!\}$
for a multiset and $c_u^{(l)}$ for the 1-WL colour of node $u$ after $l$ rounds.

\paragraph{The stacked aggregation input.}
We first make the canonical order in \cref{eqn:ldna-update} precise. The paper's
prose---that the sorted messages ``are always placed in the same positions in the
stacked messages''---makes the argument of $\psi_{\bm{\theta}}$ a fixed-size
\emph{slotted} object rather than a variable-length sorted list. Concretely,
$\big[\mathbf{m}_{\pi^*(v),u}^{(l+1)}\big]_{v\in\mathcal{N}_u}$ is the matrix
$\mathbf{S}_u\in\mathbb{R}^{N\times d_h}$ whose row $p\in\{0,\dots,N-1\}$ is
\begin{equation*}
  (\mathbf{S}_u)_p=
  \begin{cases}
    \mathbf{m}_{v,u}^{(l+1)} & \text{if } \pi^*(v)=p \text{ for some } v\in\mathcal{N}_u,\\[2pt]
    \mathbf{0} & \text{otherwise.}
  \end{cases}
\end{equation*}
Since the slot $p$ occupied by a neighbour $v$ is its canonical position
$\pi^*(v)$, reading a slot recovers the identity of the neighbour placed there.
This is exactly the information a permutation-invariant $\bigoplus$ discards, and
the mechanism LDNA restores.

\begin{theorem}
\label{thm:ldna-strict}
Let $\pi^*$ be induced by a canonical labelling: an isomorphism-invariant
bijection $\pi^*_{\mathcal{G}}:\mathcal{V}\to\{0,\dots,n-1\}$ ($n=|\mathcal{V}|$)
under which isomorphic graphs receive identical labelled graphs. Then LDNA
(\cref{eqn:ldna-message,eqn:ldna-update}) is strictly more expressive than the
1-WL test, and hence than any MP-GNN with permutation-invariant aggregation
\citep{Xu2019GIN,Morris2019WL}.
\end{theorem}

The same argument establishes the corresponding statement for LDNA-e against the
edge-coloured 1-WL refinement, by reading $\mathbf{e}_{v,u}$ into every message.

\begin{proof}
We exhibit (Part~1) parameters making LDNA at least as discriminative as 1-WL and
(Part~2) parameters separating a pair of graphs that 1-WL cannot; by
\cref{def:expressiveness} the two together give strictness.

\paragraph{Part 1: at least as expressive as 1-WL.}
We choose parameters so that, for every layer $l$, the embedding
$\mathbf{h}_u^{(l)}$ is an injective function of the 1-WL colour $c_u^{(l)}$,
arguing by induction on $l$. At $l=0$ both equal the input feature
$\mathbf{x}_u$. Assume the claim at layer $l$. Because all quantities range over
finite sets, the message map $\phi_{\bm{\Theta}}^{(l+1)}$ can encode the pair
$(\mathbf{h}_v^{(l)},\mathbf{e}_{v,u})$ injectively into
$\mathbf{m}_{v,u}^{(l+1)}$. The aggregation $\psi_{\bm{\theta}}^{(l+1)}$ is free
to ignore the slot positions and sum the rows of $\mathbf{S}_u$; since a sum of
suitable injective encodings is injective on multisets of bounded size
(\citealp{Xu2019GIN}, Lemma~5), $\psi_{\bm{\theta}}^{(l+1)}(\mathbf{S}_u)$ can
realise an injective function of
$\{\!\{(\mathbf{h}_v^{(l)},\mathbf{e}_{v,u})\mid v\in\mathcal{N}_u\}\!\}$. The
update $\gamma_{\bm{\Theta}}^{(l+1)}$ has an explicit self-argument
$\mathbf{h}_u^{(l)}$, so it can injectively combine the node's own colour with
this aggregate; an empty neighbourhood is harmless, as the self-argument then
determines $\mathbf{h}_u^{(l+1)}$ on its own. The result is an injective function
of $\big(c_u^{(l)},\{\!\{c_v^{(l)}\mid v\in\mathcal{N}_u\}\!\}\big)$, which is the
argument of the injective 1-WL update; hence $\mathbf{h}_u^{(l+1)}$ is an
injective function of $c_u^{(l+1)}$. Finally, a READOUT summing injective
encodings of the final embeddings is injective on the colour histogram
\citep{Xu2019GIN}, so whenever 1-WL assigns two graphs different colour
histograms LDNA assigns them different graph embeddings.

\paragraph{Part 2: strictly more expressive than 1-WL.}
Let $\mathcal{G}_1=C_6$ be the $6$-cycle and $\mathcal{G}_2=C_3\sqcup C_3$ two
disjoint triangles, both with a single constant node feature and no edge
features.

\emph{(i) 1-WL cannot separate them.} Both are $2$-regular with identical initial
colours, so by induction all nodes keep one common colour: if every node has
colour $a$ at round $l$, each has neighbour multiset $\{\!\{a,a\}\!\}$ and the
1-WL hash returns a single colour at round $l+1$. The two graphs thus have equal
colour histograms at every round and are 1-WL-indistinguishable.

\emph{(ii) A canonical labelling.} A canonizer fixes one representative labelling
per isomorphism class; take the cyclic order $0,1,\dots,5$ along $C_6$ and the
consecutive triangles $\{0,1,2\},\{3,4,5\}$ for $C_3\sqcup C_3$, so
$\pi^*(v)=v$.

\emph{(iii) Parameters that separate them.} On a single channel set
$\phi_{\bm{\Theta}}\equiv 1$, so every message equals $1$ and
$(\mathbf{S}_u)_p=1$ precisely when the neighbour at canonical position $p$ lies
in $\mathcal{N}_u$. Let $\psi_{\bm{\theta}}$ be the shallow per-slot linear map
with weight $2^{p}$ on slot $p$ --- slots being indexed by the \emph{global}
canonical position, per the stacked form above, not by the rank within the
neighbourhood; then
\begin{equation*}
  \psi_{\bm{\theta}}(\mathbf{S}_u)=\sum_{v\in\mathcal{N}_u}2^{\pi^*(v)} .
\end{equation*}
Reading off the neighbourhoods gives the node-level multisets
\begin{equation*}
  C_6:\ \{\!\{34,5,10,20,40,17\}\!\},\qquad
  C_3\sqcup C_3:\ \{\!\{6,5,3,48,40,24\}\!\};
\end{equation*}
for instance node $0$ of $C_6$ has neighbours at slots $1$ and $5$, giving
$2^1+2^5=34$, whereas node $0$ of $C_3\sqcup C_3$ has neighbours at slots $1$ and
$2$, giving $2^1+2^2=6$. The two multisets differ ($34$ occurs only in the
first), even though their element-sums coincide ($126$ each), so a plain
summation aggregator would not suffice and the slot-weighting is what separates
them.

\emph{(iv) Conclusion.} Taking $\gamma_{\bm{\Theta}}$ and the READOUT injective as
in Part~1, the graph embedding is an injective function of the node-level
multiset above; as the two multisets differ, LDNA assigns $\mathcal{G}_1$ and
$\mathcal{G}_2$ different embeddings. Since 1-WL cannot separate them, LDNA is
strictly more expressive than 1-WL, and hence than any MP-GNN with
permutation-invariant aggregation.
\end{proof}

\begin{remark}[The canonical labelling is essential]
\label{rem:canonical-essential}
The extra power comes from $\pi^*$ individualizing nodes, not from the stacked
form alone. If the aggregation input were reduced to the sorted sequence of
message \emph{contents} (ties left unbroken), that sequence is a canonical
encoding of the multiset $\{\!\{\mathbf{m}_{v,u}\}\!\}$, so $\psi\circ\mathrm{sort}$
is a multiset function and the model collapses to exactly 1-WL. The same collapse
occurs for any order computed from 1-WL-invariant node quantities, which fold
into the colours; and on vertex-transitive 1-WL-hard pairs---the $4\times4$
rook's graph versus the Shrikhande graph---every invariant order ties all nodes,
leaving $\bigoplus$ symmetric. An individualizing canonical labelling, obtainable
by completing the feature sort through individualization--refinement
\citep{McKay2014Nauty} (with quasi-polynomial worst-case cost
\citep{Babai2016GI}), is what supplies the node identities that break these ties.
\end{remark}

\begin{remark}[Random initialization]
\label{rem:random-init}
Injectivity on a finite domain is generic: for each map involved, the parameter
settings that make two given distinct inputs collide form a measure-zero set, and
there are finitely many input pairs. With parameters drawn independently from a
continuous distribution, every layer map is therefore almost surely injective,
and then each embedding $\mathbf{h}_u^{(l)}$ \emph{refines} the colour
$c_u^{(l)}$: the embedding determines the colour, though a position-aware random
$\psi_{\bm{\theta}}$ may split colour classes further. Refined histograms
separate whatever the colour histograms separate, so a randomly initialized LDNA
attains at least 1-WL discriminative power almost surely, without any training.
The strictness in Part~2 instead relies on explicit deterministic parameters and
needs no randomness; this complements random node initialization
\citep{Abboud2021RNI}, which randomizes the inputs rather than the aggregator.
\end{remark}

\begin{remark}[Permutation invariance]
\label{rem:invariance}
Although $\pi^*$ orders the nodes, the model output is a deterministic graph
invariant: the canonical labelling depends only on the isomorphism class, and any
choice within an automorphism orbit yields the same labelled graph---hence the
same stacked inputs $\mathbf{S}_u$ and the same output. The sort that secures
permutation invariance and the labelling that supplies node identities are
therefore one and the same object.
\end{remark}
